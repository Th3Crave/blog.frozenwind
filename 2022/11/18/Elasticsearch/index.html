Elasticsearch

2022-11-18 | technology

Elasticsearch 是一个分布式、RESTful 风格的搜索和数据分析引擎。

Kibana 是为 Elasticsearch设计的开源分析和可视化平台。

ES中的数据都是来自于MySQL , 用ES的目的不是来持久化数据的，而是因为它的数据检索、复杂数据分析的效率极高，用它来完成检索、分析的功能。

基本概念

服务端提供了 rest风格 api提供访问，简单来说，可以通过http请求访问 es

Index(索引)

type(类型) == table

document : == 一条数据（json格式的）

  1. document的 key == mysql一条记录的 列（字段）
  2. document的 value == mysql每个字段的值
